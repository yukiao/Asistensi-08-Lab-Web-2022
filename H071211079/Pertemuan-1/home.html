Welcome to My Blog

Website ini bertujuan untuk mengerjakan tugas praktikum web yang terdiri dari tampilan home, about, dan education. Tampilan home sendiri berisi tentang pengenalan blog serta tujuan blog dibuat. Tampilan about berisi tentang biodata praktikan dan disertai fotonya.

Pada website ini saya tidak menggunakan style CSS ataupun framework CSS, murni hanya munggunakan html saja. Jadi untuk teman teman yang penasaran dengan diri saya teman-teman bisa membuka halaman about, dan apabila teman-teman ingin mengetahui riwayat pendidikan saya bisa membuka halaman education di bawah ini.
