Peran joki game yaitu memainkan hingga peringkat atau level tertentu akun game sang penyewa. Joki game ini biasanya diisi orang yang ahli di bidang game. Mereka disebut sebagai pemain game profesional. Selain piawai dalam memainkan game jenis tertentu, kemampuan lain yang dimiliki yaitu mampu memainkan game hingga berjam-jam. Kemampuan ini diganjar dengan bayaran yang menggiurkan. Umumnya, para joki game ini mengiklankan diri di layanan forum game dan percakapan. Tak jarang juga, mereka menggunakan media sosial untuk mempromosikan keahlian mereka. Jasa unik ini telah mulai merebak sejak gencarnya game bergenre MMOPRG, Nexia pada 2001. Fenomena itu makin merebak setelah kemunculan game Ragnarok Online dan RF Online. Hingga kini, bisnis joki game online masih merebak. Salah satunya lewat jasa game mobile seperti Mobile Legends.
